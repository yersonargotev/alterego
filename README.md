## Getting Started

### Prerequisites

- Node.js v18+
- pnpm

### Install

```bash
pnpm install
```

### Development

```bash
pnpm dev
```
Visit [http://localhost:3000](http://localhost:3000)

### Build

```bash
pnpm build
```

### Lint & Format

```bash
pnpm lint
pnpm format
```
